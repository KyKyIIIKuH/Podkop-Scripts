# Podkop-Scripts

Набор скриптов для упрощения работы с **Podkop** и автоматизации
управления серверами.

## 📦 Возможности

-   Автоматическое обновление подписки
-   Перезапуск Podkop после обновления
-   Проверка интернет-соединения
-   Автоматическое переключение сервера при отсутствии интернета

------------------------------------------------------------------------

# ⚙️ Установка CRON

Добавьте задачу в `crontab` для автоматического обновления подписки
каждый день.

``` bash
crontab -e
```

Добавьте строку:

``` bash
# Обновление подписки с перезапуском Podkop
0 0 * * * /etc/subs.sh restart

# Проверяем наличие соединения
*/1 0 0 * * * /etc/check-connection.sh restart
```

------------------------------------------------------------------------

# 📜 Скрипты

## subs.sh

Скрипт для обновления подписки.

Функции:

-   Получает список серверов из подписки
-   Обновляет конфигурацию Podkop
-   Может перезапускать Podkop

Пример запуска:

``` bash
bash subs.sh restart
```

------------------------------------------------------------------------

## check-connection.sh

Скрипт проверки интернет-соединения.

Функции:

-   Проверяет наличие доступа в интернет
-   Если интернет отсутствует:
    -   автоматически переключает сервер в **selector**
    -   восстанавливает соединение

Пример запуска:

``` bash
bash check-connection.sh
```

------------------------------------------------------------------------

# 🧠 Логика работы

1.  `subs.sh`
    -   получает сервера из подписки
    -   обновляет конфигурацию
2.  `check-connection.sh`
    -   проверяет интернет
    -   если соединения нет → переключает сервер
3.  `cron`
    -   автоматически обновляет подписку каждый день
